# Performance Standards

Requirements:
- Lighthouse score above 90
- optimized images
- lazy loading
- minimal layout shift

---

# Optimization Rules

Use:
- code splitting
- dynamic imports
- responsive images

Avoid:
- oversized dependencies
- blocking scripts
- large video assets
