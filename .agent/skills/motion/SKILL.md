# Motion Rules

## Motion Philosophy

Motion should:
- guide attention
- reinforce hierarchy
- create depth
- feel smooth

Avoid:
- distracting motion
- excessive bouncing
- long delays

---

# Allowed Motion

- fade
- stagger
- blur reveal
- hover lift
- parallax
- opacity transitions

---

# Performance Rules

Prefer:
- transform animations
- opacity animations

Avoid:
- layout thrashing
- expensive repaints
