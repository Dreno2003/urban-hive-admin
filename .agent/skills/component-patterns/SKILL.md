# Component Patterns

## Button

Supports:
- variants
- loading state
- icon support
- accessibility

---

# Card

Cards should:
- support hover state
- maintain depth
- use soft borders

---

# Navbar

Navbar should:
- remain sticky
- support mobile navigation
- support backdrop blur

---

# Section Container

Rules:
- centered layout
- max-width system
- responsive padding
- When implement ui for any component make sure to use shadcn/ui components. If a component you want to use is not there install it
